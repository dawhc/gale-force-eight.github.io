# 机器学习——梯度下降法

  1. 方法原理

一种寻找最优解的方法，通过迭代实现

设$\theta=[\theta_j]$为参数(或者参数矩阵)，$J(\theta)$为代价函数

则目标为$\min\limits_\theta{J(\theta)}$

梯度下降法即每次沿着$J(\theta)$的梯度下降的方向更新参数$\theta$，使得$J(\theta)$的值不断下降直至局部最优。

迭代公式为：$\theta_j:=\theta_j-\alpha\frac{\partial}{\partial\theta_j}J(\theta)$

优点：容易实现；n很大时效率优秀

缺点：需要进行迭代；需要选取合适的学习率$\alpha$；容易陷入局部最优解而非全局最优解

  1. 算法伪代码

$repeat\ \ until\ \ convergence{\\ \ \ \ for\ \ \theta_j\ \ in\ \ \theta:\\ \ \ \ \ \ \ \ temp_j=\theta_j-\alpha\frac{\partial}{\partial\theta_j}J(\theta)\\ \ \ \ for\ \ \theta_j\ \ in\ \ \theta:\\ \ \ \ \ \ \ \ \theta_j = temp_j\}$
